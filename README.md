## ⚡ ProNetra – Network Controller Interface 👁️

> A precision-focused tool designed for learning network scanning and socket-based communication.

## ⚙️ Installation

```bash
pkg update && pkg upgrade
pkg install python git

git clone https://github.com/yourusername/ProNetra.git
cd ProNetra
python3 ProNetra.py
```


```bash
python3 ProNetra.py
```
